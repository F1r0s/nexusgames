Common Questions Before You Submit

My download isn't working

First, make sure you have completed the quick verification task fully — the download unlocks only after the task is confirmed. If the page stalls, try refreshing and completing the task again. Make sure your browser isn't blocking pop-ups, and that you're using a supported browser like Chrome or Firefox. Still stuck? Submit a ticket below and include the game name.

The verification task won't complete

Verification tasks require you to fully follow the partner's instructions — usually downloading an app and running it briefly, or completing a short form. Partially completing a task won't unlock your download. If you believe you completed it correctly and it still hasn't unlocked, please use real device data (not VPN) and try again. Tasks are region-specific, so availability may vary.

How do I install a modded APK on Android?

To install a mod from Mod Vault on Android: go to Settings → Security → Install Unknown Apps and enable it for your browser. Then download the APK file, tap it in your notifications or file manager, and select Install. If you're asked to uninstall the original game first, do so — some mods won't work alongside the official version. For a full step-by-step guide, visit our Game Mode & Modding Guide.

Response time & what to include

We aim to respond to all tickets within 24 hours. To help us resolve your issue faster, please include: the name of the game you're trying to download, your device type and Android/iOS version, a description of what happened, and any error messages you saw. The more detail you provide, the quicker we can help.
